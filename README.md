## ✅ Các chức năng đã hoàn thiện (Update: 01/09/2025)

1. 🔑 Đăng nhập/Đăng ký (Email/Google/Facebook)  
2. 📰 Feeds (Xem & tương tác bài đăng của mình và mọi người)  
3. 👥 Bạn bè (Danh sách, gợi ý, gửi/nhận lời mời, hủy kết bạn)  
4. 💬 Nhắn tin realtime (gửi/xóa tin nhắn)  
5. 🙍‍♂️ Profile (chỉnh sửa thông tin, avatar, nội dung quan tâm)  
6. 🔔 Thông báo (like, comment, post mới, chấp nhận bạn bè)  
7. 📰 Tin tức (tích hợp **NewsAPI**, link ngoài tới bài viết)  
8. ✍️ Tạo bài đăng (ảnh/video, ẩn/xóa/tải xuống)  
9. 🔎 Khám phá (post dạng Grid, Top Trending)  
10. ❤️ Bình luận, Like, Báo cáo  
11. 🎨 Giao diện (theme-color, đa ngôn ngữ)  
12. 🏷️ Phân loại bài viết (Keyword Matching), Feed cá nhân hóa  
13. 🗺️ Tích hợp **Map API** (tìm bạn bè xung quanh)  
14. 🛠️ Quản trị 
15. ☁️ Triển khai Hosting
